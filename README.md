# Sobre a atividade

 
 O programa em C solicita que o usuário escolha dois números inteiros, lê esses números e compara para determinar o maior entre eles. Se houver diferença, exibe o maior número e a diferença entre eles. Em seguida, o programa encerra sua execução.
